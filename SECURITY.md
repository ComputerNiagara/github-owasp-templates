|---------|-----------|

## Reporting a Vulnerability

If you discover a security vulnerability in these workflows, please do not
open a public issue. Instead, use GitHub's private vulnerability reporting:

Repo → Security → Report a vulnerability

We will respond within 5 business days and work with you to address the issue
before public disclosure.
